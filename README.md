# Music Store 

The idea is for you to clone this project 

## Steps

### Clone the repo 

```markdown
git clone https://github.com/lukepadiachy/music-store.git
```

### Check The Branches

**`cd`** into the music store directory

In order to list branches type in:

```markdown
git branch -a
```

### Change Branch 

next use this :

```markdown
git checkout starting-point
```

Now you ready to go :)
